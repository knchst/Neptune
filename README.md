### 課題3 - 「実装において工夫したポイントを記述」
課題アプリでは、アーキテクチャにMVPを採用し現在のクラシルアプリになるべく近い動きを目指して実装しました。通信部分に関しては`ishkawa/APIKit`を利用しレスポンスデータの保存は`Realm`を利用しました。
お気に入りの機能は`Realm`のNotificationを利用しデータソースの変更が即座に画面に反映されるように実装しました。
